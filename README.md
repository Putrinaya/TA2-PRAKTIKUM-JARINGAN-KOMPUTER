Tujuan lab ini adalah untuk memahami cara:
- Menyusun topologi jaringan dasar.

- Menguji **konektivitas antar perangkat** menggunakan perintah `ping`.
- Menggunakan perintah `show` untuk menampilkan informasi routing dan interface.

Lab ini juga melatih pemahaman perintah dasar **Cisco IOS** seperti:
- `no ip domain-lookup`
- `enable secret`
- `banner motd`
- `interface configuration`
- `show ip interface brief`
